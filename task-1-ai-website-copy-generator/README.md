# Task 1: AI Website Copy Generator

## Business Chosen
Taste of Mzansi – A local restaurant in Pretoria that specializes in traditional South African cuisine.

## Tool Used
Lovable AI

## Project Objective
The goal of this task was to use AI prompt engineering to generate website-ready copy for a local business.

The generator produced:
- Homepage copy with a strong value proposition
- Service page content
- Persuasive call-to-action sections

## Prompt Logic
The prompt instructs the AI to act as a professional website copywriter. It includes details about the business type, target audience, tone, and location to guide the AI in generating relevant and engaging website content.

The output is structured into:
- Homepage content
- Services page description
- Call-to-action sections
